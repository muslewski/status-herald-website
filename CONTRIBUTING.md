# Contributing to status-herald-website

This repository is the **marketing one-pager** for [`status-herald`](https://github.com/muslewski/status-herald).

## Where to file what

| Topic | Repo |
|---|---|
| Tool bugs, CLI features, install failures | **[status-herald](https://github.com/muslewski/status-herald/issues/new/choose)** |
| Questions & ideas about the tool | **[status-herald Discussions](https://github.com/muslewski/status-herald/discussions)** |
| Website copy / layout / performance | Issues on **this** repo |

## Dev (website)

```bash
npm install
npm run dev      # Vite
npm run build
```

Please keep product install commands accurate (`npm install -g status-herald` / `npx …`).
