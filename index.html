The draw

Curtain cards. Clear signals. One CLI.

Raise the curtain

Install the CLI, wire hooks, arm the session. Click any line to copy.

setup
$ npm install -g status-herald ⎘
#1 Install globally — bins: herald · status-herald
$ herald curtain install ⎘
#2 Wire Claude + Grok hooks (compat settings)
$ herald curtain arm ⎘
#3 Arm the current tmux session
$ herald curtain doctor ⎘
#4 Verify hooks, tmux, settle health
$ npx status-herald curtain install ⎘
#1 Wire hooks without a global install
$ npx status-herald curtain arm ⎘
#2 Arm this session
$ npx status-herald curtain doctor ⎘
#3 Run the checklist
$ git clone https://github.com/muslewski/status-herald.git ⎘
#1 Clone the product repo
$ cd status-herald && npm install && npm link ⎘
#2 Link herald from source
$ herald curtain install && herald curtain arm ⎘
#3 Wire + arm
✓ Need Grok native hooks too? herald curtain install grok
